# AICP Branding and Canonical Terminology

## Canonical name

**AICP — Agent Interaction Content Protocol**

## Canonical tagline

“An open content-layer protocol for agent-to-agent interaction with enforceable policies.”

## Abbreviation

- Short form: **AICP**

## Preferred key terms

- agent-to-agent protocol
- content-layer protocol
- enforcer / enforcement
- policies and attestations
- profiles and negotiation

Use these terms consistently in docs, templates, and SDK materials to avoid drift.
